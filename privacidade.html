Política de Privacidade — SonorHub

Última atualização: 27/06/2026

O SonorHub é um aplicativo pessoal de produtividade (briefing diário de voz e utilidades) desenvolvido pela FacincaniTech (André Luiz Facincani Pezzarine). Esta página explica quais dados o app acessa, para quê, e como eles são tratados.

Contas conectadas (Google, Microsoft/Outlook, Spotify)

O SonorHub permite conectar contas de terceiros para trazer informações pessoais já existentes do usuário (agenda, e-mails, contatos) pro briefing diário. O acesso é sempre via login oficial (OAuth) do próprio provedor — o SonorHub nunca vê ou armazena sua senha.

  • Google: agenda (somente leitura, pra mostrar compromissos do dia), contatos (somente leitura, pra avisar aniversários), YouTube (somente leitura, usado pela busca/reprodução de música e vídeo) e envio de e-mail (usado exclusivamente pelo botão de SOS, pra notificar um contato de confiança com a localização do usuário).
  • Microsoft/Outlook: leitura de e-mails, pra mostrar um resumo no briefing.
  • Spotify: leitura de músicas/podcasts salvos (recurso em desenvolvimento).

Esses dados são buscados em tempo real diretamente dos servidores do respectivo provedor (Google/Microsoft/Spotify) e exibidos no app — não ficam armazenados em nenhum banco de dados do SonorHub além do necessário pra manter a sessão logada.

Localização (GPS)

A localização do dispositivo é usada apenas quando o usuário aciona o recurso de SOS (segurança pessoal) ou outros recursos que dependem explicitamente de localização (como previsão do tempo). A localização nunca é coletada em segundo plano sem uma ação direta do usuário.

API de Dados do YouTube

A funcionalidade de busca/reprodução de música e vídeo (MultiSonor) usa a API de Dados do YouTube para buscar artistas, vídeos e playlists por nome. A reprodução acontece sempre dentro do player oficial embutido do YouTube (IFrame Player API) — o SonorHub não baixa, armazena nem extrai mídia do YouTube. Resultados de busca podem ser armazenados em cache no servidor por um período limitado (de 12 horas a 30 dias, dependendo do tipo de busca) para reduzir o número de chamadas repetidas à API — esse cache é só de metadados públicos (títulos, miniaturas, IDs), não de dados pessoais do usuário.

Pagamentos (Premium)

A assinatura Premium é processada pelo Mercado Pago. O SonorHub não armazena dados de cartão de crédito — todo o processamento de pagamento acontece diretamente nos servidores do Mercado Pago, seguindo a política de privacidade deles.

Dados armazenados pelo SonorHub

O app guarda localmente no dispositivo (e, quando logado, sincronizado com nosso backend) preferências e configurações do usuário — tema, contas conectadas, favoritos de música/vídeo/rádio, status da assinatura Premium. Não vendemos, alugamos nem compartilhamos esses dados com terceiros para fins de publicidade.
